Node es la parte servidor de Javascript

Node permite crear, modificar, borrar archivos del server!, asi no nesesitamos a PHP
Node y NPM son el famoso entorno dev

Instalar node

Basicamente descargamos la ultima version de la pagina: https://nodejs.org
abrimos nuestro gestor de carpetas (caja en ubuntu) como root desde la terminal: sudo caja
luego copiamos, combinamos y reemplazamos el contenido de lo descargado en /usr/
fin de la instalacion!

Inicializamos node

creamos un archivo: myfirst.js
luego lo corremos:
node myfirst.js
y listo, tenemos nuestro propio servidor! podemos verlo localmente con: localhost:8080 o desde cualquier lugar del mundo con:
ip:8080

Modulos

para incluir un modulo, por ejemplo el http, hacemos
var http = require('http');

crear modulos

crear un archivo modulo_propia.js:
